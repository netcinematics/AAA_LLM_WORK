# AAA_LLM_WORK
This repository is for LLM Training execution tasks. For educational purposes and improvement of AI skills - in RL, RLHF.

### IDE SETUP:

Go to the Extensions Marketplace (Ctrl+Shift+X) and install:

Code Runner (by Jun Han): The most important tool. It adds a "Play" button to run snippets of almost any language instantly.

Python (by Microsoft).

Quokka.js (Optional but great): For JavaScript/TypeScript, it shows the output inline next to the code as you type.

___________________________________

### The "Speed" Workflow

Don't save files. Use "Untitled" tabs or a scratchpad.

Ctrl + Shift + E (Explorer) (AAA)

New File: Ctrl + N

Set Language: Ctrl + K, then M (Type "Python", "Java", "JS" to switch mode).

Run: Ctrl + Alt + N (Run Code).
